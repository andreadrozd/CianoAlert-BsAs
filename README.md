## Alerta satelital de cianobacterias
## Buenos Aires
### Actualizaciones semanales
Semanalmente se procesan imágenes Sentinel-2 y Sentine-3 estimando retrodispersión de infrarrojos por células tipo bacterianas, pigmentos fotosintéticos
y pigmentos complementarios de cianobacterias. A partir de la integración de las variables se estima indicadores de nivel de riesgo y tendencia semanal
